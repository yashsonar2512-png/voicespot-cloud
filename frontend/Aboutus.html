// Our Story

Built by Developer,
For the Real World

VoiceSpot was born from a passion for making voice interfaces smarter, faster, and more accessible — without relying on the cloud.

"We believe voice should be a first-class interface for every application — not an afterthought."

VoiceSpot is an open-source keyword spotting system designed to run entirely on your local machine. No subscriptions, no cloud processing, no privacy concerns. Just accurate, real-time voice detection powered by deep learning. Our mission is to make spoken command interfaces accessible to every developer, researcher, and hobbyist.

Project Timeline
Phase 1
Research & Dataset Preparation
Explored the Google Speech Commands dataset. Selected 8 high-frequency keywords and preprocessed audio into standardized 1-second clips at 16kHz.
Phase 2
Model Training
Built and trained the CNN model using mel-spectrograms as input. Achieved 95%+ validation accuracy using data augmentation and early stopping.
Phase 3
Backend API Development
Built the Flask REST API with predict, enroll, verify, analytics, and history endpoints. Integrated SQLite for persistent logging and speaker storage.
Phase 4
Frontend Dashboard
Designed and built the real-time web interface with live waveform, confidence rings, keyword chips, and history table. Fully responsive across devices.
